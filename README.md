# 📚 Smart Study Planner

A modern full-stack web application designed to help students manage their study tasks, notes, schedules, and productivity efficiently using a clean and interactive dashboard.
c
## 🚀 Features

- ✅ Task Management (Create, update, track study tasks)
- 📅 Weekly Planner (Organize schedule from Monday to Sunday)
- 📝 Notes System (Subject-based notes with search & tags)
- ⏱ Timer & Stopwatch (Track study sessions)
- 🔔 Alarm System (Set study reminders)
- 📊 Analytics Dashboard (Track productivity, streaks, and progress)
- 🌙 Light / Dark Mode Toggle
- 🎯 Focus Mode for distraction-free study

## 🎨 UI / Design Highlights

- Glassmorphism (liquid glossy UI design)
- Smooth animations and transitions
- Responsive dashboard layout
- Modern sidebar navigation
- Clean and user-friendly interface

## 🛠 Tech Stack

- **Backend:** Python (Flask)
- **Database:** SQLite
- **Frontend:** HTML, CSS, JavaScript
- **ORM:** SQLAlchemy
